# Naija JUG Bootcamp

This repository serves as the official codebase for the **Naija Java User Group (JUG) Bootcamp**.

## About the Bootcamp

The Naija JUG Bootcamp is an intensive training program designed to equip aspiring and experienced Java developers with practical skills and knowledge in modern Java development. This bootcamp covers a range of topics, from core Java concepts to advanced frameworks and best practices.

## Repository Contents

This repository will host:

*   **Code Examples:** Practical code snippets and mini-projects demonstrated during the bootcamp sessions.
*   **Assignments:** Hands-on assignments and exercises to reinforce learning.
*   **Resources:** Supplementary materials, links, and documentation relevant to the bootcamp topics.

## Getting Started

To get started with the code in this repository, you will need:

*   **Java Development Kit (JDK) 17 or higher**
*   **Apache Maven** (for building and managing projects)

### Running the Example Code

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/NaijaJugAssignment.git # Replace with actual repo URL
    cd NaijaJugAssignment
    ```
2.  **Compile and Run (Maven):**
    You can compile and run the example Java applications using Maven:
    ```bash
    mvn compile exec:java
    ```
